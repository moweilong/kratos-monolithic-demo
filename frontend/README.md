# 后台管理系统

## 安装使用

- 启用Corepack

```bash
corepack enable
```

- 安装依赖

```bash
pnpm install
```

- 调试运行

```bash
pnpm dev
```

- 构建打包

```bash
pnpm build
```

- 查看ESlint

```bash
eslint --inspect-config
```
